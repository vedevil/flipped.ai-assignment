How to run the app:
1.App provides interface to select language and type the desired text.
2.On clicking submit button, It opens a new page showing suggestions.
3.Suggestions page has 4 component:
	-	Drop menu: Showing suggestions of the text in the selected language.
	-	Save: To save the result.
	-	Done: To see the result history what has been searched since last clearance.
	-	Clear: It clears the history of recent searches.
